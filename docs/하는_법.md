# 한-중 가계부 앱 — 하는 법 (처음부터 따라하기)

---

## ★ 가장 쉬운 방법 (추천)

1. **Node.js 설치**  
   https://nodejs.org 에서 **LTS** 버전 다운로드 후 설치 (다음 다음 누르면 됨).

2. **실행**  
   프로젝트 폴더 `han-cn-budget-app` 안에 있는 **`실행.bat`** 파일을 **더블클릭**한다.  
   - 처음이면 자동으로 `npm install` 하고, 브라우저가 열리면서 앱이 뜬다.  
   - 주소가 **http://localhost:5173** 이다.

3. **로그인·가족·거래**를 쓰려면 아래 "3단계: Supabase 설정"을 한 번만 하면 된다.  
   (Supabase 안 해도 **대시보드·설정·환율** 화면은 볼 수 있음.)

---

## 1단계: 프로그램 설치 확인

### 1-1. Node.js 있는지 확인

1. **명령 프롬프트** 또는 **PowerShell**을 연다.
2. 아래를 입력하고 Enter 친다.

   ```
   node -v
   ```

3. `v18.xxx` 또는 `v20.xxx` 처럼 **버전 번호가 나오면** OK.  
   "인식할 수 없습니다" 라고 나오면 → [Node.js 공식 사이트](https://nodejs.org/)에서 **LTS 버전** 다운로드 후 설치한다.

---

## 2단계: 프로젝트 실행 (Supabase 없이 화면만 보기)

Supabase 설정은 나중에 하고, **일단 화면이 뜨는지** 확인하려면:

1. **Cursor** 또는 **VS Code**에서 프로젝트 폴더 `han-cn-budget-app` 을 연다.
2. 터미널을 연다. (상단 메뉴 **터미널 → 새 터미널**)
3. 아래를 **한 줄씩** 입력한다.

   ```
   cd C:\Users\han_jisoo\han-cn-budget-app
   npm install
   npm run dev
   ```

4. 터미널에 `http://localhost:5173` 같은 주소가 나온다.
5. **브라우저**에서 그 주소로 들어간다.  
   → "한-중 가계부" 화면이 보이면 성공이다.  
   (이때는 아직 로그인/가족 기능은 안 되고, **대시보드·설정·환율** 같은 화면만 볼 수 있다.)

---

## 3단계: Supabase 설정 (로그인·가족·거래 저장이 되게 하기)

### 3-1. Supabase 가입 및 프로젝트 만들기

1. 브라우저에서 **https://supabase.com** 접속한다.
2. **Start your project** → **Sign in with GitHub** (또는 이메일) 로 **가입/로그인** 한다.
3. 로그인 후 **New Project** 를 누른다.
4. 다음처럼 입력한다.
   - **Name**: `han-cn-budget` (아무 이름이나 가능)
   - **Database Password**: 본인이 기억할 비밀번호 입력 (꼭 메모해 둔다)
   - **Region**: 가까운 지역 선택 (예: Northeast Asia)
5. **Create new project** 를 누르고, 1~2분 기다린다.

### 3-2. 프로젝트 주소와 비밀키 복사하기

1. Supabase 왼쪽 메뉴에서 **Project Settings** (톱니바퀴) 클릭.
2. 왼쪽에서 **API** 를 클릭.
3. 아래 두 개를 **복사** 해 둔다.
   - **Project URL** (예: `https://abcdefgh.supabase.co`)
   - **anon public** 키 (긴 문자열, **Project API keys** 안에 있음)

### 3-3. 데이터베이스 테이블 만들기 (SQL 실행)

1. Supabase 왼쪽 메뉴에서 **SQL Editor** 를 클릭.
2. **New query** 를 누른다.
3. 프로젝트 안에 있는 **`docs/SUPABASE_SCHEMA.sql`** 파일을 연다.  
   (Cursor에서 `han-cn-budget-app` → `docs` → `SUPABASE_SCHEMA.sql`)
4. 그 파일 **전체 내용**을 복사해서 Supabase SQL Editor **창에 붙여넣기** 한다.
5. 아래쪽 **Run** (또는 실행 버튼)을 누른다.  
   → "Success" 나 오류 없이 끝나면 OK.

### 3-4. 실시간 반영 켜기 (Realtime)

1. Supabase 왼쪽 메뉴에서 **Database** → **Publications** 로 간다.
2. **supabase_realtime** 이 보이면 클릭한다.
3. **Add table** 같은 버튼으로 **transactions** 테이블을 추가한다.  
   (테이블 목록에서 `transactions` 체크 후 저장하면 됨.)

### 3-5. 앱에 주소와 키 넣기 (.env)

1. 프로젝트 폴더 `han-cn-budget-app` 안에 **`.env`** 파일을 만든다.  
   (이미 `.env.example` 이 있으면, 그걸 복사해서 이름을 `.env` 로 바꿔도 된다.)
2. `.env` 파일을 열고 아래처럼 **한 줄씩** 넣는다.  
   (`여기에_넣기` 부분을 3-2에서 복사한 값으로 바꾼다.)

   ```
   VITE_SUPABASE_URL=여기에_Project_URL_넣기
   VITE_SUPABASE_ANON_KEY=여기에_anon_public_키_넣기
   ```

3. 저장한다.

---

## 4단계: 앱 다시 실행해서 로그인·가족·거래 쓰기

1. 터미널에서 **Ctrl + C** 로 `npm run dev` 를 멈춘다.
2. 다시 실행한다.

   ```
   npm run dev
   ```

3. 브라우저에서 **http://localhost:5173** 로 연다.
4. 아래 **거래내역** 탭으로 간다.
   - **로그인 / 가입**: 이메일과 비밀번호 입력 후 "가입" 또는 "로그인" 클릭.
   - **가족 만들기**: 로그인 후 "새 가족 만들기" → 표시 이름(예: 남편) 입력 후 만들기.
   - **초대 코드로 참여**: 다른 사람은 "초대 코드로 참여" → 만든 사람이 Supabase **families** 테이블에서 `invite_code` 를 알려주면, 그 코드 입력 후 참여.
5. 가족이 만들어진 뒤에는 **지출/소득** 금액과 통화(원/위안) 입력 후 "추가" 하면 된다.
6. **대시보드** 탭에서 일/주/월 합계와 차트를 볼 수 있다.
7. **설정** 탭에서 **기준 통화(원화/위안)** 를 바꿀 수 있고, **로그아웃** 할 수 있다.

## 자주 하는 질문

- **"가입했는데 이메일 안 와요"**  
  Supabase 대시보드 → **Authentication** → **Providers** → **Email** 에서 "Confirm email" 를 끄면, 인증 없이 바로 로그인 가능하다.

- **"초대 코드 어디 있어요?"**  
  Supabase **Table Editor** → **families** 테이블 → 만든 가족 행의 **invite_code** 칸에 있는 8자리 코드다.

- **"npm install 할 때 오류 나요"**  
  터미널을 **관리자 권한**으로 열거나, `node -v` 가 18 이상인지 확인한다.

---

## 요약 순서

1. `node -v` 로 Node 확인 → 없으면 Node.js 설치  
2. `cd han-cn-budget-app` → `npm install` → `npm run dev` 로 화면 확인  
3. Supabase 가입 → 프로젝트 생성 → API에서 URL·anon 키 복사  
4. SQL Editor에서 `SUPABASE_SCHEMA.sql` 실행  
5. Realtime에 `transactions` 테이블 추가  
6. 프로젝트에 `.env` 만들고 URL·키 넣기  
7. `npm run dev` 다시 실행 → 브라우저에서 로그인·가족 만들기·거래 입력  

여기까지 하면 "하는 법" 은 끝이다.
